Get nodes by type

ApiKeyAuth
query Parameters

required
string

concise
boolean

/search

Search for nodes matching a query

query Parameters
query
required
string

Search query string

limit
integer
Default: 25

Maximum number of results to return

concise
boolean
Default: false

Only include name and file in returned data

node_types
string
Example: node_types=Page,Function,Class

Comma-separated list of node types to filter by

Local development server

http://localhost:3000/search

ApiKeyAuth
query Parameters
node_type
string

Reference ID of the node to start from (alternative to node_type+name)

integer

direction
string
